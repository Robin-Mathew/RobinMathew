My Projects

Online Dictionary

Built an Online Dictionary Website using React and NodeJS, hosted on IBM Cloud to give out meanings of words entered with pronunciation. Words were parsed and indexed using Solr and stored in MongoDB. Two components provided Word of the day and Trending word based on popular search.

Technology Used: React.js, NodeJS, IBM Cloud, Solr, MongoDB, Swagger

Movie Ticket DApp

Developed a Movie Ticket Collection DApp using Solidity hosted on Netlify where users can generate and buy or sell their ticket tokens using MetaMask and transfer it between accounts. A front-end site was built for buying/selling using ReactJS where users can view other's tickets and buy it.

Technology Used: Solidity, Metamask, Netlify, React.js

Augmented Reality Games for Children with Autism

Developed games for autistic kids using via Unity 3D and C#. The games were controlled using hand gestures detected by infrared rays using the Leap Motion device.

Technology Used: Unity 3D, Leap Motion, C#,
